企业内部系统Enterprise
CMS 发稿系统、素材库、OA 审批、企业 IM
现有系统MCP 适配器API 网关
接口对接 / 适配层开发
  • 通过 MCP 适配层将现有系统接入 AI 编排服务，不重建系统
  • 每个内部系统开发一个 MCP Server 适配器，暴露标准化接口
  • 例：CMS MCP Server 提供"创建稿件""查询栏目""提交审核"等能力
  • 素材库 MCP Server 提供"按标签检索""按日期归档""批量重命名"等能力
  • 适配层遵循"新增系统只需开发适配器，无需修改编排核心"的扩展原则
核心原则：不自研大模型，工程精力集中在编排、权限治理、企业系统集成。每一层可独立演进，模型供应商可在不修改业务代码的情况下切换。
